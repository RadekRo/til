# ternary operator 
#### exists in python as a single line "if"

Usage: `return a if a > b else b`.\
This will return a or b according to true/false if result. 

__Example:__

```
a = 1
b = 2
return a if a > b else b
```
// Returns:
``` 2 ``` according to *False* if condition.
