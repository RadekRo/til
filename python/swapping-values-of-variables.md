# swapping values of variables
#### inplace swap

```
a = 2
b = 3

a, b = b, a
print(a, b)
```
Output: `3, 2`\
Values of the two variables swapped!

You can add modifications during a swap, like this:
```
a, b = b, a + 2
print (a, b)
```
Output: `3, 4`

Pretty cool, huh?
